Compiling the source code:
Before compiling the source code in CodeBlocks, some settings must be tweaked first. 
In CodeBlocks, go to settings>compiler. Under compiler settings, click "linker setting" and then paste "-lwinmm" in the textbox under "other linker options." After this, click "Ok."
